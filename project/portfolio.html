My Portfolio

After realizing how out-of-date my original portfolio was, I started thinking about how I could show off my skills building a better portfolio. My original portfolio was made using Foundation 4 and some hacky CSS, in other words there was room for improvement.

This project was made to accurately reflect my front end developement skills. Here's the technology I'm using currently:

  • CSS Grid
  • CSS Pre/Post-processors (SASS, Minify)
  • Flexbox

Knowing that I wrote all of the stylesheets for my first portfolio in vanilla CSS gives me anxiety. I learned a lot about SCSS shortly after finishing it so I never got the chance to start from scratch. This project would not have been completed without SCSS since I rely on it for how the Grid stylesheet works.
